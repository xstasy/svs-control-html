# svs-control-html
Pure in-browser javascript/html control of 
* SVS SB1000PRO
* SVS 3000 Micro

Could work with other SVS subwoofers too.

Should work on any device web browser that supports:

[Web Bluetooth API](https://caniuse.com/web-bluetooth)

Can be installed as PWA on phone/desktop
[https://ftown.cloud/svs/](https://ftown.cloud/svs/)

Or download repo and open svs.html in browser.


Based on [pySVS](https://github.com/logon84/pySVS) by [logon84](https://github.com/logon84)
